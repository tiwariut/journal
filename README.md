# Journal

> A blog application.

## Usage

Rename ".env.example" to ".env" and update the values/settings to your own.

## Install Dependencies

```
npm install
```

## Run App

```
# Run in dev mode
npm run dev
```

## Database Seeder

To seed the database with data from the "\_data" folder, run

```
# Destroy all data
node seeder -d

# Import all data
node seeder -i
```
